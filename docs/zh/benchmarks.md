# 基准

::: info **关于**
比较库的最佳方式是在**您自己的环境**中使用**您自己的工作负载**运行基准测试。每个项目都有独特的需求——延迟、吞吐量、内存使用和集成复杂性——没有任何单一的测试能够覆盖所有情况。

我建议您将 `uuid` 与其他库一起测试，并选择最适合您需求的工具。
:::
